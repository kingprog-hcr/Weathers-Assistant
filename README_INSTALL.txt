WeatherProgramm : Installation

1. Obtenez une clé API gratuite sur https://openweathermap.org/api
2. Créez un fichier .env dans ce dossier avec le contenu suivant :
   API_KEY=votre_cle_ici
   GROQ_API_KEY=votre_cle_groq_ici  (optionnel : pour l'enrichissement IA)

3. Lancez WeatherProgramm

Linux/Mac : double-cliquez ou lancez ./WeatherProgramm dans le terminal
Windows   : double-cliquez sur WeatherProgramm.exe

Note : une connexion internet est requise pour la météo.
